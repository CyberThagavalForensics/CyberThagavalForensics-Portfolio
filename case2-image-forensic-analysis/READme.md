# Case 2: Image Authenticity Check

## Objective
To verify whether a given image is original or has been modified.

## Files Used
- Original image
- Edited image

## Method
The images were visually examined and compared using forensic tools to check for any signs of manipulation or editing.

## Findings
The edited image showed visible differences when compared with the original image, indicating that the image had been modified.

## Conclusion
The investigation confirms that the edited image is not the same as the original and shows signs of manipulation.
